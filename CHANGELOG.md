# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-07

### Added

- Initial release of the Terraform GCP Cloud SQL module
- Cloud SQL instance with PostgreSQL and MySQL support
- High availability configuration
- Read replica support with independent sizing
- Private Service Networking for private IP
- Automated backup with point-in-time recovery
- Database and user management
- Query Insights integration
- Basic and complete usage examples
